Background Music Setup
======================

To add background music to the reward page:

1. Replace 'background-music.mp3' with your own music file
2. Keep the filename as 'background-music.mp3' or update the reference in reward.html
3. Supported formats: MP3, OGG, WAV (MP3 recommended for best compatibility)
4. Recommended: Use a romantic song that has special meaning for you
